# UCUG1903

课程演示与 **UCUG1903_Proj.pptx** 配套（C 大调自然音列自 **低 sol 196 Hz** 至 **高 mi 659.25 Hz** 共 13 级；**中音 mi = 329.63 Hz**）：可多单元顺序播放；每单元可选 **0.2～1.0 s**（步长 0.1）、音高、方波/锯齿、可选噪声；全程波形与 WAV。

## Web

路径：`web/`，可直接打开 `index.html`。

### 操作

- **单元数量**（1～16）+ **编辑单元**：**持续时间**（0.2–1.0 s，步长 0.1）、**音高**、**波形**、**叠加噪声**。
- **叠加噪声（近似）**：可选，与主波形同段叠加；低通随当前音高变化；与主音两路时音量按 **√2** 缩放以防削顶。
- **依次播放 / 停止 / 重置当前单元 / 导出 WAV**；**整体响度** 滑块调节每段基准音量。
- **单元总谱**：横轴时间（块宽∝时长），纵轴为唱名行（每音一行，钢琴卷帘式）；**蓝色=方波**，**绿色=锯齿**；**黄褐斜线+描边=叠噪**；**蓝框=当前编辑单元**。

### 数据

合成事件仍使用 `freqHz`、`durSec`、`startSec`；见 `web/song-schema.js`。
